# Meus Estudos de JavaScript 🚀

**Repositório dedicado às minhas aulas práticas e exercícios da linguagem.**

Este projeto contém os códigos que estou criando enquanto aprendo **JavaScript voltado para o Back-end** através de videoaulas no YouTube. Ao invés do tradicional JS no navegador, o foco aqui é dominar a linguagem pura rodando os códigos diretamente no terminal utilizando o ambiente do **Node.js**.

## 🛠️ Tecnologias e Perfil
- **Linguagem:** JavaScript (JS)
- **Runtime:** Node.js
- **Minha Base/Background:** Ciência da Computação (c/ C puro e Estruturas de Dados).

## 🚀 Como executar os códigos
Para testar qualquer exercício deste repositório, basta ter o Node.js instalado na máquina, abrir o terminal na raiz desta pasta e rodar o comando `node`:

```bash
# Criei no formato: node <nome-do-arquivo>
node calculadoraAniversario.js
```

## 🎯 Objetivo da Trilha
Consolidar minha adaptação da tipagem estática e gerenciamento de ponteiros do `C` para a tipagem dinâmica e *Garbage Collector* do ecosistema JS. 

O objetivo do repositório é construir uma ponte de conhecimento sólida para, no futuro próximo, **migrar para o TypeScript (TS)** no Node.js!
