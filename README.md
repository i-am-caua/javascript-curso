# Meus Estudos de JavaScript 🚀

**Repositório dedicado às minhas aulas práticas e exercícios da linguagem.**

Este projeto contém os códigos que estou criando enquanto aprendo **JavaScript voltado para o Back-end** através de videoaulas no YouTube. Ao invés do tradicional JS no navegador, o foco aqui é dominar a linguagem pura rodando os códigos diretamente no terminal utilizando o ambiente do **Node.js**.

## 🛠️ Tecnologias e Perfil
- **Linguagem:** JavaScript (JS)
- **Runtime:** Node.js
- **Minha Base/Background:** Ciência da Computação (c/ C puro e Estruturas de Dados).

## 📁 Estrutura do Projeto
Para facilitar a revisão no futuro e manter uma ordem cronológica dos estudos, os códigos estão categorizados por módulo/assunto:

*   **`01-fundamentos/`**: Variáveis (`let/const/var`), tipos de dados e operadores básicos.
*   **`02-logica-e-estruturas/`**: Exercícios de Algoritmos envolvendo Condicionais (`if/else`), Laços de Repetição (`for`/`while`) e Manipulação de Vetores (`Arrays`).
*   **`03-funcoes-e-objetos/`**: Práticas de encapsulamento, closures, *Arrow Functions* e lógica Orientada a Objetos.

## 🚀 Como executar os códigos
Para testar qualquer exercício deste projeto, basta ter o Node.js instalado na máquina, abrir o terminal na raiz da pasta e rodar o comando `node` com o caminho da pasta:

```bash
# Formato: node <pasta>/<nome-do-arquivo.js>
node 02-logica-e-estruturas/calculadoraAniversario.js
```

## 🎯 Objetivo da Trilha
Consolidar minha adaptação mental da tipagem estática do `C` para a tipagem dinâmica do ecossistema JS. O objetivo macro deste repositório é construir uma ponte de conhecimento prática e sólida para, no futuro próximo, **migrar para o TypeScript (TS)** como foco principal de Back-end!
